# IFMIS Dashboard

## 📌 Project Description

This is a React-based dashboard application with a login system.

## 🚀 Features

* User login using useState
* Welcome message after login
* Reusable InfoTile components
* Dashboard with multiple data tiles
* useEffect for side effects

## 🛠️ Technologies Used

* React.js
* JavaScript
* HTML & CSS
